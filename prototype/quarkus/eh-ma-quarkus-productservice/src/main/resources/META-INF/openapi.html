ProductService API (1.0.0)

Download OpenAPI specification:Download

API specification for the ProductService of the e-shop application. Manages digital products, including creation, updates, deletion, and retrieval. Provides search and pagination capabilities.

Product Management

Endpoints related to managing digital products.

List all products with optional search and pagination

Authorizations:
bearerAuth
query Parameters
page
integer >= 1
Default: 1

Page number for pagination (default is 1)

limit
integer [ 1 .. 100 ]
Default: 10

Number of items per page (default is 10)

search
string
Example: search=photo editor

Search term to filter products by title or description

category
string
Example: category=Software

Filter products by category

priceMin
number <float>
Example: priceMin=10

Minimum price to filter products

priceMax
number <float>
Example: priceMax=100

Maximum price to filter products

createdAfter
string <date-time>
Example: createdAfter=2023-01-01T00:00:00Z

Filter products created after a specific date

createdBefore
string <date-time>
Example: createdBefore=2023-12-31T23:59:59Z

Filter products created before a specific date

sort
string
Enum: "priceAsc" "priceDesc" "dateAsc" "dateDesc" "titleAsc" "titleDesc"
Example: sort=priceAsc

Sort order (e.g., 'priceAsc', 'priceDesc', 'dateAsc', 'dateDesc')
